A step by step walk through to the your first html page

HTML is shor for Hyper Text Markup Language.Basically,it's the "code" behind every webpage- even this one.If you're just beginning to learn HTML, let us tell you that it' s a fairly easy task . HTML, without styling,can't do anything more than setting a layout,drawing a table, or creating frames-but it is handy as it helps you structure the content correctly, which is important whwn you sit down to style to your HTML.

However simple this might seem,it is a mighty useful tool when it comes to full-fledged wed development. Various tools easily eliminate the HTML coding from your work process-but if you want to be full control of your web-page, you'll need to have some command over HTML.

Through this article,we aim to give you the essential HTML building blocks that'll help you get up and running.Reading this,you'll be able to understand an HTML source code and even modify it for your own good!

Step One - TAGS

Tags are what you'll see the most when you look at any HTML source .A tag can ideally be seen as a wrapper to any item on your HTML document. Tags tell what magic is to be done on the content enclosed by them.

Let's look at two types of tags:

  1. <tag-example-1>I need a closign tag<tag-example-1>
  2. <tag-example-2>I don't need a cloding tag.

In the first example, the sentense is wrapped by two tags.The first one is called the opening tag and the second one is closing tag.Everything in between is affected by properties of the tag. very commonly used example of such tags are<html>,<head>,<body>,<strong>,etc.

The second example tags about loner tags- as in,they don't need a closing tag to function.Although it's not required these type of tags are often written as to make the debugging of code easier Common example of such tags are <hr/>- used for horizontal line,<br/>- to break the line,etc.

Step Two - HTML,HEAD, and BODY:The three pillars of your document

These tags are essential for any HTML document.They parcel out the significant parts of your HTML code.

  • <BODY></BODY> is palced below your <HEAD> tag, and Everything that you want to be displayed on your screen comes under this tags.Text,images,links, and pretty much anything you can see in your browser live inside this tags
  • <HTML></HTML> wrap your entire code.Everyting else in your HTML document needs to come inside these tags.
  • <HEAD></HEAD> includes things like title,style,and scripts.Haed is usually present at the top(hah!) just inside the</HTML>tag.

Step three -A few tags that'll make your page pretty

Now that you know how to set up the skeleton of your document,let's proceed with the things that will go inside your<BODY>tag and do some magic!

Some basic text formatting tags:

  • <b></b> make your text lool bold
  • <i></i> makes you write incursive
  • <u></u>underlines what you just wrote

For example, this piece of code

      <html>
        <head></head>

Should produce something like this on your browser:Don't fret too much about the <br/>.It's just for breaking the line so that you can start from the next line.Enter key does little when it comes to changing lines in your HTMl document.

Tags to help you structure your content:

  • <br/> breaks the line, making you continue to the next line
  • <p> stands for paragraph. it divides your content into paragraphs

Note: you need to use these tags as spce and enter keys do very little when it comes to formatting content inside an HTML document

Heading Tags:

HTMl provides you with six tags, from <H1></H1>to <H6></H6> to help you create different sized headers quickly

Inserting an Image:

All that's good, but what fun without images on the webpages? Don't worry, <IMG/> to the rescue!The image tag has a mandatory attribute called "source".Basically,it tells the browser where it should look for the image. The syntax goes something like:

<img sre = "path_to_your_image"/>

Furthermore,it also has attribute like height and width that let you specify the height and width you want your image to take.

Lists:

HTMl has two types of lists - ordered and unordered.EAch item of your list has to be enclosed in a tag.The syntax for creating a list is fairly simple.

Suppose you want to create a list like:

  • Item 1
  • Item 2
  • Item 3

The following code will easily do the job for you:

    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
     </ul>

This,by the way, was an example of an unordered list.For an ordered list,all you need ot do is replace <ul> with <ol> and </ul>with </ol>

Let's see what the following code does:

        <html>
          <head>  </head>
          <body>
             <ul>
              <li>I am unordered list's item 1! </li>
              <li>I am unordered list's item 2! </li>
              <li>I am unordered list's item 3! </li>
              <li>I am unordered list's item 4! </li>
             </ul> 
             <ol>
             <li>I am ordered list's item 1! </li>
             <li>I am ordered list's item 2! </li>
             <li>I am ordered list's item 3! </li>
             <li>I am ordered list's item 4! </li>
            </ol> 
          </body>
        </html>
